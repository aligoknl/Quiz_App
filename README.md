## 5. Must Haves

-   A user should click and choose an option for each question.  
-   A user should take a feedback if the selected answer is correct or wrong.
-   A user should be able get a help from the external links or see the correct answer of each question
-   A user should be see her/his progress and the time on every question.
-   A user should be able to see her/his result on the result page.
-   A user should be able to restart the app on result page.

## 6. Nice-to Have

-   A user can see a message on the result page according to her/his performance.
